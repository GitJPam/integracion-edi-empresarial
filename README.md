# Integración EDI Empresarial

Simulador de procesamiento de archivos EDI utilizado en entornos empresariales para integraciones críticas.

---

## Objetivo

Demostrar el flujo completo de una integración EDI:

* Lectura de archivos
* Validación de estructura y reglas de negocio
* Transformación a modelo de dominio
* Persistencia en sistemas internos

---

## Características

* Arquitectura limpia (Clean Architecture)
* Separación de dominio, aplicación e infraestructura
* Reglas de negocio centralizadas
* Persistencia simulada en SQL Server
* Código orientado a sistemas reales de alto volumen

---

## Casos de uso

* Integraciones corporativas
* Facturación electrónica
* Plataformas financieras
* Sistemas transaccionales empresariales
